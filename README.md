# Aventuras no Maranhão 🎮  

**Aventuras no Maranhão** é um jogo educativo desenvolvido com Construct 2, cujo objetivo é conscientizar os jogadores sobre a importância da preservação do patrimônio histórico e cultural de São Luís e do Maranhão, criado para projeto de conclusão de curso do curso de Sistemas de Informação do IFMA. O projeto utiliza elementos interativos e lúdicos para despertar o interesse pela história e cultura da região.

---

## 📂 Estrutura do Repositório  

Este repositório contém:  
- **Projeto Construct 2**: Arquivos do jogo desenvolvidos no Construct 2.  
- **Ilustrações**: Imagens e assets criados no Adobe Illustrator.  
- **Mídias**: Áudio, música e outros recursos utilizados no jogo.  

> **Observação**: O jogo exportado e deployado está versionado em um repositório separado. Consulte o repositório abaixo para obter a versão final jogável:  
> [**aventurasnomaranhaogame**](https://github.com/alexiakidd/aventurasnomaranhaogame)

## 🖇️ Contribuições  

Este projeto é **open source**, e o repositório público permite que qualquer pessoa colabore para melhorias ou expansões. Sinta-se à vontade para explorar o código, propor ajustes e enviar *pull requests*.  

Para contribuir, siga os passos abaixo:  
1. Faça um fork deste repositório.  
2. Crie uma *branch* para suas alterações: `git checkout -b minha-branch`.  
3. Realize as modificações e faça um commit: `git commit -m "Descrição das alterações"`.  
4. Envie a *branch* para o repositório: `git push origin minha-branch`.  
5. Abra um *pull request* para análise.
